# Minhas Finanças

Aplicativo simples de controle financeiro pessoal, feito apenas lançar receitas, despesas, transferências e cartões de credito com suas despesas relacionadas, ainda estou melhorando e implementando outras funcionalidades que para mim, são essenciais. 

## 🚀 Tecnologias Utilizadas

*   **Backend:** Java 21 com Spring Boot (API REST).
*   **Frontend:** Angular com Angular Material para UI/UX.
*   **Banco de Dados:** PostgreSQL 17 (neon.tech).
*   **Infraestrutura Local:** Docker Compose
